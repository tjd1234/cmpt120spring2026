# Turtle Graphics Examples

- [shapes.py](shapes.py) (draws a variety of shapes, including a flower and some
  patterns)
- [rectangle.py](rectangle.py) (draws a rectangle with a given width and height)
- [polygon.py](polygon.py) (draws a regular polygon with a given number of
  sides)
- [polyline.py](polyline.py) (draws a polyline, i.e. a series of connected lines
  segments at a given angle)
- [arc.py](arc.py) (draws a circular arc with a given radius and angle)
- [row_of_squares.py](row_of_squares.py) (draws a row of squares with a given
  number of squares and length)
- [grid.py](grid.py) (draws a grid of squares with a given number of columns and
  length)
