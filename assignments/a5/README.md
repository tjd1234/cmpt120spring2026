# Assignment 5: Lights Out and Tic Tac Toe

In this assignment your task is to implement **one of** either [Tic Tac
Toe](TicTacToe/) or [Lights Out](Lightsout/). See those links for what to do and
how they will be marked.

Each project is worth the same amount of marks. You should look both over and
then decide which one you want to implement.

## Submitting Your Work

When you are done, you'll either have a file called `ttt.py` or `lightsout.py`.
Whichever one you have, please submit it on Canvas

Each program will be marked according to its marking scheme (each is worth the
same amount of marks).
